Educational attempt at creating a basic webpage from scratch.

Written in Python 3.6 using Flask.

Developed on AWS Cloud 9 and will be deployed with AWS EC2 / Elastic Beanstalk.

Repository name: "web_practice"

Repository URL: https://github.com/danharris-stu/web_practice.git

Only optimized for 1920 x 1080 currently, smaller window sizes and mobile devices may not appear as intended.

You need to install the 'Flask' package to run this server from your own machine.

To do so on linux use the following command:
sudo pip install Flask
